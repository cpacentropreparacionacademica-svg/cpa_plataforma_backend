# Módulo societario

Módulo NestJS migrado desde routers Express. Expone únicamente los recursos definidos originalmente para `societario`.

Recursos expuestos:
- `clase-titulo` → `societario.clase_titulo`
- `dividendo` → `societario.clase_titulo`
- `dividendo-pago` → `societario.dividendo`
- `emision-titulo` → `societario.clase_titulo`
- `tenencia` → `societario.emision_titulo`
- `titular` → `persona.persona`
- `transferencia-titulo` → `societario.emision_titulo`
