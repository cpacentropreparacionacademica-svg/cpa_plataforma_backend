# Módulo inventario

Módulo NestJS migrado desde routers Express. Expone únicamente los recursos definidos originalmente para `inventario`.

Recursos expuestos:
- `bien` → `contabilidad.cuenta`
- `bien-instancia` → `inventario.bien`
- `bien-lote` → `inventario.bien`
- `movimiento-detalle` → `inventario.bien`
