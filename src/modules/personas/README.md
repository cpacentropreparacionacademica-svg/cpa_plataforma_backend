# Módulo personas

Módulo NestJS migrado desde routers Express. Expone únicamente los recursos definidos originalmente para `personas`.

Recursos expuestos:
- `estudiante` → `persona.persona`
- `estudiante-padre` → `persona.persona_padre`
- `padre` → `persona.persona_padre`
- `proveedor` → `persona.proveedor`
- `tutor` → `persona.persona`
- `unidad-educativa` → `persona.unidad_educativa`
- `usuario` → `persona.persona`
