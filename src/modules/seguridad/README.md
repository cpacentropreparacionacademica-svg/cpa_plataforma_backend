# Módulo seguridad

Módulo NestJS migrado desde routers Express. Expone únicamente los recursos definidos originalmente para `seguridad`.

Recursos expuestos:
- `permiso` → `seguridad.permiso`
- `rol` → `seguridad.rol`
- `rol-permiso` → `seguridad.rol`
- `usuario-permiso` → `persona.persona`
- `usuario-rol` → `persona.persona`
