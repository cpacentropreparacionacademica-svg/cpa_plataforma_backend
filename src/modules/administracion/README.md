# Módulo administracion

Módulo NestJS migrado desde routers Express. Expone únicamente los recursos definidos originalmente para `administracion`.

Recursos expuestos:
- `departamento` → `administracion.departamento`
- `empleado` → `persona.persona`
- `empleado-posicion-pago` → `administracion.empleado`
- `empleado-registro-pago` → `administracion.empleado_registro_pago`
- `kpi` → `administracion.kpi`
- `objetivo-kpi` → `administracion.kpi`
- `posicion` → `administracion.posicion`
