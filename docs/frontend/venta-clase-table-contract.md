# Frontend table contract - Parte de clases pasadas

Pantalla sugerida: tabla editable con columnas:

- Fecha
- Hora ingreso
- Hora salida
- Nombre completo estudiante
- Tutor
- Motivo clase
- Materia / Producto
- Tema
- Subtema
- Efectivo
- QR
- CxC
- Paq.
- Sit. Base

Endpoint principal:

```http
POST /api/contabilidad/venta-clase/registrar-batch
```
